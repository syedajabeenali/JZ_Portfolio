## 🔧 Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/syedajabeenali/JZ_Portfolio.git
cd JZ_Portfolio
```

### 2. Install Dependencies 

```bash
npm install
```

### 3. Run the app

```bash
npm run dev
```
